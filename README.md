# AI-Sahayak  
### Multi-Grade AI Teaching Assistant for Rural Classrooms

AI-Sahayak is an AI-powered teaching support platform designed specifically for multi-grade rural classrooms, where a single teacher manages students from different grade levels in the same classroom.

The system intelligently generates structured, grade-differentiated lesson plans, quizzes, worksheets, exam papers, bilingual explanations, and educational stories — reducing teacher workload and enabling effective multi-level instruction.

---

## 🚩 Problem Statement

In many rural schools:

- One teacher handles 2–3 grades simultaneously.
- Students sit together in a single classroom.
- Syllabus topics differ across grades.
- Teachers struggle to design structured multi-level lesson plans.
- Language barriers affect student understanding.

AI-Sahayak solves this by generating structured, grade-wise, classroom-ready academic content instantly.

---

## 🎯 Core Unique Value Proposition (UVP)

AI-Sahayak intelligently:

- Analyzes syllabus of multiple grades
- Detects overlapping or hierarchical topics
- Merges related topics (e.g., "Plants" & "Photosynthesis")
- Creates structured multi-grade lesson plans
- Designs parallel time-block teaching strategies if topics differ
- Generates classroom-ready content suitable for blackboard teaching

---

## ✨ Key Features

### 1️⃣ Smart Lesson Planner (Multi-Grade Optimizer)

- Teacher uploads syllabus of 2 or more grades
- AI identifies related or foundational topic connections
- Merges basic and advanced concepts intelligently
- Generates structured multi-grade teaching strategy
- Provides:
  - Time-block lesson flow
  - Grade-wise reinforcement activities
  - Blackboard writing plan
  - Learning outcomes

Example:
Grade 1 – "Plants"  
Grade 2 – "Photosynthesis"

AI suggests:
- Teach plant basics to all students
- Extend explanation to photosynthesis for higher grade
- Assign reinforcement tasks separately

---

### 2️⃣ Visual Aid Generator

- Generates multiple simple images for classroom teaching
- Content is adjusted by grade level
- Designed for easy explanation in rural classrooms
- Supports concept clarity for low-literacy environments

---

### 3️⃣ Quiz Generator

Creates classroom activity quizzes with:

- 1 common question for all grades
- Grade-specific difficulty levels
- Short and clear format
- Blackboard-friendly structure
- Answer key included

---

### 4️⃣ Worksheet Generator

Generates 3 learning levels:

- Level 1 – Basic (Weak students)
- Level 2 – Practice (Average students)
- Level 3 – Challenge (Advanced learners)

Includes:
- Mixed question types (Fill blanks, match, short answer)
- Answer key
- Simple explanations
- Independent learning support

---

### 5️⃣ Question Paper Generator

Generates real exam-format papers:

- Proper school header formatting
- Section A / B / C structure
- Balanced difficulty (Easy / Medium / Hard)
- Exact marks calculation
- Internal choices
- Printable structured layout
- No answers included

Supports:
- MCQ
- Short Answer
- Long Answer
- Fill in the Blanks
- Mixed Format

---

### 6️⃣ Knowledge Base (Bilingual Learning Support)

- Simple explanation in English
- Local language explanation (Telugu / Hindi / Tamil)
- Rural life real-world examples
- Practice question
- Designed for students with low English proficiency

---

### 7️⃣ Story Generator

Generates:

- Rural village-based educational stories
- Scene-wise structure
- Simple image descriptions
- Moral of the story
- Vocabulary list with meanings
- Comprehension questions
- Optional bilingual output

---

## 🏗 System Architecture

- Frontend: React.js + TypeScript
- Backend & Auth: Supabase
- AI Engine: Gemini API
- Styling: Tailwind CSS
- Prompt Engineering: Structured rule-based prompting system
